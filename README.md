# כלי בדיקת תוכן - PassportCard 🎯

כלי פנימי לצוות הביזנס של PassportCard לבדיקה, אישור ועריכת טקסטים לפני העלאה לייצור.

## ✨ תכונות עיקריות

- **ממשק Wizard** - מעבר בין תחומים בצורה מסודרת
- **אישור ועריכה** - כל שדה ניתן לאישור או עריכה לפני המשך
- **שמירה אוטומטית** - ההתקדמות נשמרת ב-LocalStorage
- **המשכיות** - אפשרות להמשיך מאיפה שעצרת
- **מעקב התקדמות** - פס התקדמות ואבני דרך
- **ניווט מהיר** - מיני-מפה לניווט בין תחומים
- **קיצורי מקלדת** - עבודה מהירה עם המקלדת
- **מצב כהה/בהיר** - תמיכה ב-Dark Mode
- **תמיכה מלאה ב-RTL** - ממשק עברי מלא

## 🚀 שימוש

### אופציה 1: פתיחה ישירה
פשוט פתחו את `index.html` בדפדפן.

## 🔧 איך להשתמש

1. **פתחו את הכלי** - פתחו `index.html` בדפדפן
2. **העלו קובץ JSON** - גררו קובץ JSON או לחצו לבחירה
3. **עברו על התחומים** - כל תחום מציג את השדות שלו
4. **אשרו או ערכו** - לחצו "אישור" אם הטקסט תקין, או "עריכה" לשינוי
5. **הורידו את הקובץ** - בסיום, הורידו את הקובץ המעודכן

## ⌨️ קיצורי מקלדת

| קיצור | פעולה |
|-------|-------|
| `A` | אשרו הכל בתחום הנוכחי |
| `D` | החלפת מצב כהה/בהיר |
| `?` | הצגת עזרה |
| `←` `→` | ניווט בין תחומים |
| `1-9` | קפיצה לתחום לפי מספר |
| `Escape` | ביטול עריכה |
